⚙️ Einmalige Einrichtung erforderlich
Um Google Drive Sync zu nutzen, musst du einmalig eine kostenlose Google Cloud OAuth-Client-ID erstellen und in den Code eintragen.
Schritte: 1) Gehe zu console.cloud.google.com → APIs & Services → Anmeldedaten → OAuth-Client-ID erstellen → Typ: Webanwendung → Autorisierte JS-Ursprünge: die URL dieser HTML-Datei → Client-ID kopieren. 2) Öffne diese HTML-Datei in einem Texteditor, suche nach const CLIENT_ID = '' und füge deine ID ein. 3) Datei neu laden.
